# Geometry-first four-domain paper campaign

## Decision

Determine whether the proposed paper matrix is executable and fair before
admitting its 2,268 learning-rate-selection trainings.  The immediate
falsifiable decision is narrower: can each new core pathway train, reload, and
run closed-loop evaluation from the same exact snapshot without oracle future
menus, non-finite losses, or missing artifacts?

## Human steering incorporated

The requested headline comparison contains random, token language modelling,
next-sentence modelling, next-sentence latent prediction, their recurrent
counterparts, and a geometry-constrained JEPA on four datasets.  Each learned
model--dataset--width cell uses the frozen nine-rate grid on seeds 0--2;
validation mean selects the rate, and seeds 3--4 complete the five-seed paper
estimate.  The comprehensive mechanism ablation is on iGSM, with one
ProofWriter endpoint to check that the conclusion is not purely arithmetic.
The later steering addition makes frozen-representation analysis a central
claim: the campaign now compares what JEPA and generative LM states encode,
what text fixed-capacity readouts reconstruct, and whether latent geometry
tracks counterfactual progress rather than template identity.

## Validity audit before submission

- Twenty-nine focused tests pass, including recurrence, geometric advantage
  regression, infinite padding masks, external schemas, non-oracle planning,
  and target-encoder evaluation mode.
- A recurrent token model completed the exact gate shell command on CPU,
  produced `best.pt`, reloaded it, and wrote closed-loop metrics.
- The same checkpoint exported causal pre-action states, completed grouped
  held-out linear probes, and trained a detached frozen-feature sentence
  decoder end to end.  These smoke numbers are not scientific results.
- The external adapters preserve the difference between a full action
  catalogue and the actions currently executable.  Symbolic executors and
  teacher continuations are labels/evaluation machinery, not model inputs.
- The JEPA value is constrained to `Q(s,a,g)=V(F(s,a),g)`; ranking and direct
  advantage regression supervise that same consequence geometry.
- Existing `shuffle_actions=true` settings were discovered to shuffle
  executed intents away from outcomes, not merely candidate order.  Headline
  configurations now keep alignment; a named shuffled-action negative control
  preserves the falsifier.
- ProofWriter and PlanBench adapters compile official-format records.  ALFWorld
  trace compilation exists, but full interactive collection/replay remains a
  failed admission gate.  Therefore no ALFWorld LR sweep is admissible yet.

## Scale decision

Submit only four process-gate jobs, distributed across Grünau, Alex, Lise,
and Grete.  These cover geometry JEPA and all three recurrent baseline paths.
Do not submit the paper-scale grid until all four artifacts validate and each
dataset separately passes its data admission gates.  This staging changes no
learning-rate or seed rule; it prevents a software failure from consuming a
paper-scale budget.

## Outcomes that change direction

- Any non-finite loss, reload mismatch, missing metric, or oracle-menu access:
  repair and repeat only the failed gate.
- All four gates pass: admit the first complete 27-run LR cell (nine rates by
  three seeds), then use the frozen validation selector.
- External catalogue recall below 100%, executor replay failure, or split
  overlap: block that dataset while other validated datasets proceed.
